About Archived Projects

Domain names and external APIs get obsolete over time. Some domain names don't make sense to be renewed since they're no longer in use, while some external API are no longer maintained. This is a way for me to preserve them without any constraints from these external factors.

I have taken a snapshot of every API requests from every projects that requires it and used it as data instead of actually fetching it. The load time are faked to simulate realistic loading time when it was still functional. I only commented on the fetching part instead of deleting it so you can always see how it was originally implemented.

KKN Covid Jogja

Active period: Jun 2020 - Jun 2021
Original domain: kkncovidjogja.online

This is a progressive web app that I made in early 2020 for my Community Service Program in my college, which was done online due to the Covid-19 pandemic. This website aims to educate people about the new pandemic with information such as news related to the pandemic, total cases, hoax information, as well as relevant articles made by the team.
